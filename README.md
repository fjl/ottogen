## What you'll need:
- the ['FF Dax Bold'](https://www.fontfont.com/fonts/dax/bold) font (there's no free alternative, sorry)
- [node.js](http://nodejs.org)

Once you have the font installed, run

    $ npm install
    $ node server.js
